=========
Aemanager
=========

Aemanager est une application web de gestion pour auto-entrepreneur écrit en python à l'aide du framework django.

Fonctionnalités
---------------

Il permet de :

 * gérer ses clients
 * définir des projets
 * créer des devis
 * gérer sa facturation
 * suivre ses dépenses
 * suivre son chiffre d'affaire
 * prévoir ses charges
